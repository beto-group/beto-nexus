## Overview

**Beto Nexus** is the central hub for everything in the Beto Ecosystem. While it powers the **Beto Marketplace** for one-click Datacore component installation, it is designed to be much more.

Nexus serves as your personal gateway, connecting your Obsidian vault to a world of utilities, rewards, and account-based features. Whether you are looking for the latest views or managing your Beto profile, Nexus is the key.

## Installation

### Method 1: Via BRAT (Recommended)

Since Beto Nexus is currently in beta, the easiest way to install it is using **BRAT** (Beta Reviewers Auto-update Tool).

<img alt="BRAT Installation Guide" src="https://raw.githubusercontent.com/beto-group/beto.assets/main/MARKETPLACE/beto-marketplace-installation-brat.gif" width="100%" />

1.  Install the **BRAT** plugin from the Obsidian Community Plugins list.
2.  Open the command palette (`Cmd/Ctrl + P`) and search for `BRAT: Add a beta plugin for testing`.
3.  Enter the repository URL: `https://github.com/beto-group/beto-nexus`.
4.  Click **Add Plugin**.
5.  Once installed, go to **Settings > Community Plugins** and enable **Beto Nexus**.

### Method 2: Manual Installation

1.  Download the latest release from the [Releases Page](https://github.com/beto-group/beto-nexus/releases).
2.  Extract the `beto-nexus` folder.
3.  Move the folder to your vault's plugin directory: `.obsidian/plugins/`.
4.  Reload Obsidian.
5.  Enable the plugin in **Settings > Community Plugins**.

## Usage

1.  **Open Nexus**: Click the Beto Nexus icon in the ribbon or use the command palette.
2.  **Browse Marketplace**: Visit [marketplace.beto.group](https://marketplace.beto.group) or browse directly within the plugin.
3.  **Connect Account**: (Coming Soon) Log in to access your dashboard and rewards.
